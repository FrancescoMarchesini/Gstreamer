GST ogg/Vorbis audio player command line

Questa applicazione è un applicazione sunto degli estratti di codice precedenti.
Tale App è un player audio per file audio Ogg/Vorbis

Ogg/Vorbis che cos'è?
è un formato di compressione di dati audio prodotto da Xiph.org

Che tipo di compressio è?
Ogg/vorbis alla stregua degli mp3 è una compressione con perdite, ovvero una compressione che elimina alcune
informazioni audio al fine di ottenere il miglior tasso di compressione possibile, pur mantendo il risultato
il piu vicino possibile al file originario

Differenza Dall'Mp3 e altri algoritmi di compressione:
1 è openSources
2 La codifica è dinamica ovvero il numero di bit utilizzati dipende dalla complessità del flusso dati in un dato momento
3 Registrazione polifonica, metre mp3 è stereo e quindi solo due canali
